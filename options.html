GitHub PR status in Gmail

Paste a GitHub token so the extension can read PR status. A token is required for private repositories.

How to create a token

Go to github.com/settings/tokens/new to create a classic token — the simplest option, and it works across every org you belong to. Then:

  1. Note: GitHub PR status in Gmail
  2. Expiration: 90 days
  3. Select scopes: check repo.
  4. Generate token, copy it, paste it above, and click Save.

Seeing No repo access? Make sure the repo scope is checked. If your organization enforces SAML SSO, also click Configure SSO next to the token and authorize it for the org.

🔒 This token is stored only in your browser, and is sent only to GitHub (api.github.com).
